PHPWord初使用

最近项目中使用到导出网页内容到word功能，使用php内置函数也能实现导出，但是样式比较单一，于是找到一个开源的工具 PHPWord，研究了一番，比较强大，这里介绍下使用；

###　PHPWord 要求PHP版本5.3+，因为5.3之后引入了命名空间的概念；

Requirements

  • PHP 5.3+
  • PHP Zip extension
  • PHP XML Parser extension

    Optional PHP extensions

  • PHP GD extension
  • PHP XMLWriter extension
  • PHP XSL extension
